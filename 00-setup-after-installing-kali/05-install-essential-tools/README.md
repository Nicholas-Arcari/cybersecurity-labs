# Installa strumenti essenziali

Comando:

```Bash
sudo apt install htop curl tmux net-tools
```

- Perché farlo?

Gli strumenti di default sono basilari. htop è migliore di top, curl è vitale per scaricare risorse, tmux permette di non perdere il lavoro se cade la connessione

- Cosa accade dopo?

Migliori drasticamente la tua efficienza operativa e la capacità di diagnosticare problemi

- Cosa rischi se non lo fai?

Perderai tempo prezioso in operazioni macchinose e avrai meno visibilità su cosa sta consumando le risorse del tuo computer
