# Personalizza l'ambiente desktop

Comando:

```Bash
xfce4-settings  # (o sudo apt install gnome)
```

- Perché farlo?

L'ergonomia non è vanità. Lavorare ore su terminali con contrasto sbagliato o font piccoli stanca la vista e porta a errori

- Cosa accade dopo?

Avrai un ambiente di lavoro confortevole che riduce l'affaticamento visivo e cognitivo

- Cosa rischi se non lo fai?

Minore produttività e maggiore probabilità di commettere errori di sintassi o di lettura dei log
