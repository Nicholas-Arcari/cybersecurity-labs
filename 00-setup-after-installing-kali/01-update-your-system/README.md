# Aggiorna il sistema

Comando:

```Bash
sudo apt update && sudo apt upgrade -y
```

- Perché farlo? 

Le immagini ISO di installazione sono statiche e spesso vecchie di mesi. I repository contengono le versioni più recenti del software

- Cosa accade dopo?

Il sistema scarica le ultime definizioni dei pacchetti, corregge bug noti e aggiorna il kernel Linux

- Cosa rischi se non lo fai?

Lavorerai con software obsoleto che potrebbe contenere vulnerabilità di sicurezza già note agli attaccanti, rendendo il sistema instabile o facilmente penetrabile
