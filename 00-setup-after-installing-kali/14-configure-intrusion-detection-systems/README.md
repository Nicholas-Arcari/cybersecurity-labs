# Configura IDS (Suricata)

Comando:

```Bash
sudo apt install suricata
```

- Perché farlo?

Kali (sopratutto Purple) è una distro difensiva. L'IDS (Intrusion Detection System) è la telecamera di sorveglianza della rete

- Cosa accade dopo?

Suricata analizza ogni pacchetto in transito e genera un allarme se rileva firme di attacco (es. qualcuno sta usando Nmap contro di te)

- Cosa rischi se non lo fai?

Non saprai mai se qualcuno sta scansionando o attaccando la tua macchina finché non sarà troppo tardi
