SDR (Radiofrequenza)

Questa è la sezione più avanzata. Se hai un'antenna (come una RTL-SDR da 30€ o un HackRF), qui è dove salvi i segnali radio che catturi (apricancelli, telecomandi auto, stazioni meteo).

Must-Have Tools: Universal Radio Hacker (URH) (fantastico per visualizzare e modificare un segnale radio registrato), GQRX (per ascoltare la radio).
