# Active Scanning: Port Scanning Phase

Objective: Identification of active services, port filtering and in-depth analysis of exposed configurations on the target. Tools: Masscan, Nmap

This directory documents two distinct and complementary approaches to port scanning: speed (Masscan) and depth (Nmap Scripts).

---

## Directory Structure

1. High-Speed Scanning (Masscan)

In this section, the use of asynchronous scanners for large-scale reconnaissance is explored.

- Activity: Practical comparison between Nmap and Masscan.
- Key Finding: Critical analysis of "False Negatives" generated by stateless scanners (Masscan) in virtualized environments (NAT), demonstrating the importance of cross-validation.

2. Deep Enumeration (Nmap Scripts)

In this section, the activity shifts to advanced service enumeration using the Nmap Scripting Engine (NSE).

- Activity: Scanning of the SMB service (port 445) with default and security scripts.
- Key Finding: Identification of Information Disclosure (exposed NetBIOS Name) and a critical Security Misconfiguration (SMB Signing not required), which exposes the target to SMB Relay attacks.
