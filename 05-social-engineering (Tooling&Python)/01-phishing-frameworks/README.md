Phishing Frameworks

Qui metti le piattaforme che gestiscono le campagne.

GoPhish: È lo standard professionale. A differenza degli script semplici, GoPhish ti dà statistiche dettagliate: chi ha aperto l'email, chi ha cliccato il link, chi ha inserito la password. Se vuoi fare simulazioni serie, devi usare questo.

SET (Social-Engineer Toolkit): Il "vecchio" re. È ottimo perché include vettori di attacco multipli, non solo email (es. clonare un sito web in tempo reale).

PyPhisher/Zphisher: Sono ottimi per test veloci, ma spesso vengono bloccati subito dai filtri antispam perché usano tunnel generici (come Ngrok). Vanno bene nella sottocartella Automated-Wrappers.
